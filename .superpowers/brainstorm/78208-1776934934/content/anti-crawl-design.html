反爬防护模块设计

IP池 + UA轮换 + Cookie池 + 频率自适应 完整防护套件

反爬防护模块架构
🌐 IP轮换池
  • 支持外部代理池 API 接入
  • IP健康检测（可用性、响应速度）
  • 失败自动切换下一个IP
  • Redis缓存可用IP列表
🎭 UA轮换
  • 预置主流浏览器UA池
  • 按平台匹配UA（移动端/桌面端）
  • 每次请求随机选取
  • 支持自定义UA扩展
🍪 Cookie池
  • Redis存储登录态Cookie
  • Cookie有效性检测
  • 过期自动刷新/更换
  • 按数据源隔离Cookie
⏱️ 频率自适应
  • 默认请求间隔配置
  • 检测到限制自动降频
  • 正常后逐步恢复频率
  • 随机延迟防检测
📦 代理池服务配置示例 
# 系统配置 - 支持多种代理服务商接入
proxy_config:
  enabled: true
  provider: "custom"  # 或 "ip2world", "brightdata", "oxylabs" 等

  # 自定义代理池API
  custom:
    api_url: "https://your-proxy-service.com/api/v1/get"
    api_key: "${PROXY_API_KEY}"
    response_format: "json"  # 返回格式
    ip_field: "ip"           # IP字段名
    port_field: "port"       # 端口字段名

  # 策略配置
  strategy:
    max_failures: 3          # 单IP最大失败次数后切换
    health_check_interval: 300  # IP健康检测间隔(秒)
    pool_size: 100           # 缓存IP池大小
    timeout: 10              # 请求超时(秒)
