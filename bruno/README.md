# NewsRadar API Bruno Collection

用于管理数据源的 API 请求集合。

## 使用方法

1. 使用 Bruno 打开此目录
2. 设置环境变量 `API_URL` 和 `TOKEN`
3. 运行请求

## 目录结构

- `groups/` - 数据源组管理
- `sources/` - 数据源管理（热榜、KOL 等）

## 数据源组 ID

- 热榜组: `00000000-0000-0000-0000-000000000001`
- 微博 KOL 组: `00000000-0000-0000-0000-000000000002`
- X KOL 组: `00000000-0000-0000-0000-000000000003`
- Reddit 组: `00000000-0000-0000-0000-000000000004`
